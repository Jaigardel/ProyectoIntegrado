Rally Fotográfico

Este repositorio forma parte de un proyecto de fin de grado centrado en el desarrollo de una plataforma web para la participación fotográfica. La aplicación permite a los usuarios subir imágenes como parte de un rally fotográfico y ofrece un sistema de votación pública para valorar las contribuciones. Aquí se documentan los recursos técnicos, código fuente y herramientas utilizadas durante su desarrollo.

Etapas del Desarrollo

1. Análisis y diseño

Definición de requisitos, estructura de base de datos y arquitectura general del sistema.

2. Desarrollo de interfaz

Diseño del frontend con HTML, CSS y Bootstrap para una experiencia clara y accesible.

3. Programación backend

Implementación de lógica de negocio con PHP y conexión a MySQL mediante PDO.

4. Gestión de roles y autenticación

Sistema de login y control de accesos según tipo de usuario (invitado, registrado, admin).

5. Subida y almacenamiento de imágenes

Integración con Supabase para el almacenamiento externo de fotos y metadatos.

6. Pruebas y despliegue

Verificación de funcionalidades, corrección de errores y publicación en servidor externo.
